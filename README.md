# 📝 Simple Quiz Management System

> A simple and clean **Java Spring Boot** application for managing quiz questions with a built-in web interface.

---

## ✨ Features

- ➕ Add new quiz questions easily
- 📋 View all quiz questions in one place
- 🗄️ H2 in-memory database (no setup required)
- 🎨 Thymeleaf-powered frontend interface
- ⚡ Lightweight and fast to run locally

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| ☕ Java 21 | Core programming language |
| 🌱 Spring Boot | Backend framework |
| 🍃 Thymeleaf | Frontend templating engine |
| 🗃️ H2 Database | In-memory database |
| 🔧 Maven | Build & dependency management |

---

## 🚀 Getting Started

### ✅ Prerequisites

Make sure you have the following installed:

- Java 21+
- Maven 3.x+

### ▶️ Run the Project

```bash
mvn spring-boot:run
```

Then open your browser and navigate to:

```
http://localhost:8080
```
